# Tab Audio Recorder

Chrome拡張機能。選択したタブの音声を録音してoggファイルとして保存します。

## インストール

1. `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」でこのディレクトリを選択

## 使い方

1. 拡張アイコンをクリック → 新しいタブでレコーダーが開く
2. 「録音するタブを選択...」をクリック
3. 録音したいタブを選択し、**「タブの音声も共有する」をONにする**
4. 「共有」をクリック → 録音開始（アイコンが赤くなる）
5. 「録音停止 & 保存」をクリック → oggファイルがダウンロードされる

## 注意

- 「タブの音声も共有する」トグルをONにしないと音声が録音されません
- 録音中はアイコンをクリックしても新しいタブは開きません
- 録音ファイルはogg形式で保存されます（NotebookLM対応）

## MP3への変換

```bash
ffmpeg -i recording.ogg recording.mp3
```
